# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.1.x   | ✅        |
| < 1.1   | ❌        |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, contact the maintainer via:

- Discord: [discord.gg/jjr8WmPZ](https://discord.gg/jjr8WmPZ)
- GitHub: [@Theopote](https://github.com/Theopote) (private message if possible)

Include a description of the issue, steps to reproduce, and potential impact. You can expect an initial response within 7 days.
