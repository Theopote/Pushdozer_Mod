# 工具模式详解

## 破坏模式
1. 使用方法：
   - 按`G`键循环切换至破坏模式
   - 右键点击开始破坏区域
   - 可通过配置界面设置忽略的方块类型

2. 高级设置：
   - 在配置文件中设置`breakableBlockIds`来限制可破坏的方块
   - 使用`ignoredBlockIds`设置要忽略的方块（如花草）

## 铺设模式
1. 基础操作：
   - 按`G`键循环切换至铺设模式
   - 右键点击开始放置方块
   - 使用配置界面选择要放置的方块类型

2. 进阶技巧：
   - 配合地形生成使用
   - 使用锁定高度功能创建平整地形
   - 利用智能填充算法自动填补空隙

## 平滑模式
1. 操作方法：
   - 按`G`键循环切换至平滑模式
   - 右键拖动来平滑地形
   - 使用不同大小的范围获得不同的平滑效果

2. 使用技巧：
   - 小范围平滑保留更多细节
   - 大范围平滑创造更自然的过渡
   - 结合锁定高度功能创建特定高度的平滑地形

## 显示模式详解
1. 线框模式：
   - 清晰显示操作范围边界
   - 低性能消耗，适合低配置电脑
   - 适��精确操作和建筑对齐
   - 使用`V`键快速切换至此模式

2. 表面模式：
   - 直观显示影响区域表面
   - 适合大范围地形修改
   - 方便预览最终效果
   - 使用半透明效果展示重叠区域
   - 按`V`键切换至此模式

3. 无显示模式：
   - 最优性能表现
   - 适合熟练用户快速操作
   - 用于录制视频时去除视觉干扰
   - 再次按`V`键切换至此模式

4. 模式切换技巧：
   - 在精确建造时使用线框模式
   - 地形塑造时使用表面模式
   - 熟悉操作后可使用无显示模式
   - 根据电脑性能选择合适模式
