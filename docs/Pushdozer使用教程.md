## 简介

Pushdozer 是一个强大的 Minecraft Fabric 地形编辑模组，让地形编辑变得像推土机一样简单高效。它提供了直观的多模式地形操作、丰富的笔刷几何体、灵活的标高系统，以及服务端权威的多人同步与撤销/重做功能。

### 主要特性
- **多种工作模式**：挖掘、铺设、平滑、表面处理等
- **8种笔刷几何体**：球体、长方体、圆柱体、圆锥体等
- **4套标高系统**：无限制、跟随玩家、锁定、自定义
- **撤销/重做**：每名玩家30步操作历史
- **多人优化**：服务端权威执行，批处理网络广播

## 安装与配置

### 系统要求
- Minecraft: 1.21.4
- Fabric Loader >= 0.16.9
- Fabric API
- Java >= 17

### 安装步骤
1. 安装 Fabric Loader 与 Fabric API
2. 将 Pushdozer 模组 JAR 文件放入 `.minecraft/mods` 文件夹
3. 启动游戏（客户端或专用服务器）

### 获取工具
- **创造模式**：在 `Tools` 分组中找到 `推土机`
- **生存模式**：根据配方合成（查看合成表）

## 基本操作

### 打开配置界面
- 手持推土机工具
- 按 `P` 键（默认）或设置的自定义按键
- 或在控制设置中绑定 `打开配置` 按键

### 基本使用流程
1. **选择工作模式**：在配置界面中选择所需的工作模式
2. **配置笔刷形状**：选择几何体类型并调整参数
3. **设置操作距离**：调整最大操作距离
4. **配置标高**：选择合适的标高限制模式
5. **执行操作**：右键点击目标位置

### 显示模式
- **线框模式**：显示笔刷轮廓的线框
- **点云模式**：显示笔刷范围内的点云
- **表面模式**：半透明面预览，便于观察笔刷覆盖范围
- **隐藏模式**：不显示任何预览

## 工作模式详解

### 1. 挖掘模式 (EXCAVATE)
**功能**：清除指定范围内的方块
- **分层挖掘**：启用后可分层挖掘
- **挖掘层数**：设置一次挖掘的层数
- **方块选择**：选择允许被破坏的方块类型

### 2. 铺设模式 (PLACE)
**功能**：在指定范围内铺设方块
- **自适应生物群系**：根据生物群系自动选择合适的地表方块
- **选择自然方块**：手动选择要铺设的方块类型
- **支持方块**：草方块、泥土、耕地、沙子、石头等

### 3. 平滑模式 (SMOOTH)
**功能**：平滑地形表面
- **平滑强度**：0-1之间，0保留原始形状，1完全重塑
- **提升平滑**：向上平滑地形
- **降低平滑**：向下平滑地形
- **自适应平滑**：智能选择平滑方向

### 4. 表面粗糙 (SURFACE_ROUGHEN)
**功能**：为地形表面添加粗糙纹理
- **粗糙强度**：控制粗糙程度
- **噪声频率**：控制纹理细节程度
- **噪声持久性**：控制多层噪声的相对强度
- **噪声八度数**：设置噪声层数
- **自动缩放**：根据笔刷半径自动调整参数

### 5. 表层转换 (SURFACE_CONVERT)
**功能**：将地表方块转换为其他类型
- **多方块混铺**：支持多种方块按比例混合
- **百分比配置**：为每种方块设置出现概率
- **均匀分配**：快速平均分配所有方块概率

### 6. 骨粉生长 (BONE_MEAL)
**功能**：对植物使用骨粉促进生长
- 无需额外配置
- 自动识别可生长的植物

### 7. 批量种植 (BATCH_PLANT)
**功能**：批量种植植物
- **种植类型**：树木、花朵、草类、自定义
- **种植密度**：控制植物分布密度
- **团簇大小**：控制植物聚集程度
- **遵守生物群系**：只在适合的生物群系中种植
- **树种选择**：橡树、云杉、白桦、丛林树等
- **花组选择**：平原花卉、森林花卉、全部花卉等

### 8. 水岸处理 (SHORELINE_PROCESS)
**功能**：处理水域边缘的地形
- **水岸类型**：沙滩、堤岸、自适应、泥泞、岩石
- **水岸宽度**：设置过渡效果宽度
- **启用植物种植**：自动种植适合的植被
- **植被密度**：控制植被种植密度
- **标高限制**：可选择在标高以上或以下操作

## 笔刷几何体

### 1. 球体 (Sphere)
- **参数**：半径
- **适用场景**：圆形区域的地形编辑

### 2. 长方体 (Box)
- **参数**：长度、宽度、高度
- **适用场景**：方形区域的地形编辑

### 3. 正八面体 (Octahedron)
- **参数**：半径
- **适用场景**：钻石形状的区域编辑

### 4. 圆柱体 (Cylinder)
- **参数**：半径、高度
- **适用场景**：圆柱形区域的地形编辑

### 5. 圆锥体 (Cone)
- **参数**：半径、高度
- **适用场景**：圆锥形区域的地形编辑

### 6. 椭球体 (Ellipsoid)
- **参数**：X半径、Y半径、Z半径
- **适用场景**：椭圆形区域的地形编辑

### 7. 正四面体 (Tetrahedron)
- **参数**：边长
- **适用场景**：三角形金字塔形状的区域编辑

### 8. 三棱柱 (Triangular Prism)
- **参数**：长度、宽度、高度
- **适用场景**：三角形柱状区域的地形编辑

## 标高系统

### 1. 标高不限 (NO_LIMIT)
- **说明**：不限制操作标高
- **适用场景**：需要自由编辑任何高度的地形

### 2. 跟随玩家标高 (FOLLOW_PLAYER)
- **说明**：操作标高会随着玩家移动
- **适用场景**：需要跟随玩家位置进行地形编辑

### 3. 锁定玩家标高 (LOCKED_ONCE)
- **说明**：锁定标高到当前玩家脚下位置
- **适用场景**：需要在固定高度进行精确编辑

### 4. 自定义标高 (CUSTOM)
- **说明**：通过滑动条自定义标高
- **适用场景**：需要在特定高度进行地形编辑

## 高级功能

### 撤销/重做系统
- **撤销**：按 `Z` 键撤销上一步操作
- **重做**：按 `Y` 键重做被撤销的操作
- **历史记录**：每名玩家保存30步操作历史
- **冷却时间**：防止频繁撤销/重做

### 方块选择系统
- **分类选择**：按类别选择方块（地形、建筑、装饰等）
- **搜索功能**：输入方块名称快速查找
- **全选/清空**：快速选择或清除所有方块

### 植物选择系统
- **分类选择**：按植物类型选择（树苗、花朵、草类等）
- **生物群系自适应**：根据生物群系自动选择合适植物
- **自定义列表**：手动选择要种植的植物

## 多人游戏

### 服务端权威
- 所有操作在服务端执行，确保一致性
- 客户端只负责界面交互和预览显示

### 网络优化
- **批处理广播**：小操作即时发送，大操作分批发送
- **光照更新**：自动处理光照和邻居方块更新
- **性能优化**：大范围操作分块同步

### 权限控制
- 默认所有玩家可用
- 内置半径上限（默认最大100）
- 可与区域保护/权限系统集成

### 配置同步
- 配置文件保存在本地 `config/pushdozer_config.json`
- 保存时会通过 C2S 同步到**服务端**，供该玩家自己的 Pushdozer 操作使用
- **不会**广播给其他玩家，也不会写入他人的本地配置文件
- 每名玩家的工作模式、笔刷形状等设置互不影响

## 常见问题

### Q: 为什么无法使用推土机？
A: 请检查：
- 是否手持推土机工具
- 是否在创造模式或拥有足够权限
- 是否安装了正确版本的Fabric API

### Q: 操作没有效果怎么办？
A: 请检查：
- 工作模式是否正确设置
- 操作距离是否足够
- 标高限制是否合适
- 是否选择了正确的方块类型

### Q: 游戏卡顿怎么办？
A: 建议：
- 降低操作半径
- 分批执行大范围操作
- 关闭不必要的显示模式
- 检查服务器性能

### Q: 如何恢复误删的地形？
A: 使用撤销功能（Z键）或重做功能（Y键）

## 性能优化建议

### 客户端优化
- 使用合适的显示模式（线框模式性能最好）
- 避免过大的操作半径
- 定期清理操作历史

### 服务器优化
- 限制最大操作半径
- 设置操作冷却时间
- 监控服务器性能

### 多人游戏优化
- 大范围操作分批执行
- 避免同时进行多个大操作
- 合理设置网络同步参数
