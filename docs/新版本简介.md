 # Pushdozer - 专业地形编辑工具

![pushdozer](https://cdn.modrinth.com/data/cached_images/8cebc3138a6e1ce2c904a0b8482809cb94a1e646_0.webp)

## ✨ 概述

Pushdozer 是一款专为 Minecraft 建造者和创作者设计的强大地形编辑模组。它让地形编辑变得像推土机一样简单高效，提供直观的多模式地形操作、丰富的笔刷几何体、灵活的标高系统，以及服务端权威的多人同步与撤销/重做功能。无论是平整山脉、挖掘隧道、批量种植，还是创建自然水岸，Pushdozer 都能让这些任务变得轻而易举。

## 🎯 核心特性

### 🛠️ 八种工作模式
- **挖掘模式** - 快速清除大面积区域，支持分方块过滤
- **铺设模式** - 智能铺设地表方块，支持自适应生物群系
- **平滑模式** - 智能地形平滑，处理陡峭边缘和粗糙表面
- **表面粗糙** - 为地形添加自然纹理和粗糙效果
- **表层转换** - 将地表方块转换为其他类型，支持多方块混铺
- **骨粉生长** - 对植物使用骨粉促进生长
- **批量种植** - 智能种植树木、花朵、草类，支持生物群系自适应
- **水岸处理** - 创建自然水岸，支持沙滩、堤岸等多种类型

### 🎨 八种笔刷几何体
- **球体、长方体、正八面体、圆柱体、圆柱体、椭球体、正四面体、三棱柱** 

### 📐 四套标高系统
- **标高不限** - 自由编辑任何高度，适合复杂地形
- **跟随玩家** - 操作标高跟随玩家位置
- **锁定标高** - 精确高度控制，适合平整地形
- **自定义标高** - 手动设置特定高度

### 🖥️ 三种显示模式
- **线框模式** - 显示笔刷轮廓，性能最佳
- **点云模式** - 显示点云预览，直观效果
- **隐藏模式** - 不显示预览，专注操作

### 🔄 撤销/重做系统
- 支持30步操作历史
- 实时保存操作记录
- 防止频繁撤销/重做的冷却机制

## 🌍 多人游戏支持

### 服务端权威
- 所有操作在服务端执行，确保数据一致性
- 客户端只负责界面交互和预览显示
- 防止作弊和异常操作

### 网络优化
- 小操作即时发送，大操作分批发送
- 自动处理光照和邻居方块更新
- 大范围操作分块同步，避免卡顿

### 权限控制
- 默认所有玩家可用
- 内置半径上限（默认最大100）
- 可与区域保护/权限系统集成

## ⚙️ 安装要求

### 系统要求
- **Minecraft**: 1.21.4
- **Fabric Loader**: >= 0.16.9
- **Fabric API**: 必需
- **Java**: >= 17

### 安装步骤
1. 安装 [Fabric Mod Loader](https://fabricmc.net/use/) (版本 >= 0.16.9)
2. 下载并安装 [Fabric API](https://modrinth.com/mod/fabric-api)
3. 从 [发布页面](https://modrinth.com/mod/pushdozer/versions) 下载最新版本的 Pushdozer
4. 将下载的 JAR 文件放入 `.minecraft/mods` 文件夹
5. 启动 Minecraft 并享受 Pushdozer！
