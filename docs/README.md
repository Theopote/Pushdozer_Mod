# Pushdozer

> Pushdozer是一个操作简单易用的地形编辑模组，为Minecraft建筑师和创作者提供了直观的工具来进行大规模地形修改。无论是平整山地、挖掘隧道，还是清除世界中特定的方块，Pushdozer都能让这些工作变得轻而易举；其最大的特点是可以像推土机一样快速平整地形，填补空地；也能平滑地形边界，使地形在视觉上更自然化。

# 特性

- 🚀 快速高效的地形编辑
- 🎨 三种编辑模式：破坏模式、放置模式、平滑模式
- 📐 两种笔刷形状：长方体模式、球形模式
- ⚡ 实时预览笔刷范围：线框预览、表面显示、无干扰模式
- 🔄 撤销/重做支持：30步撤销/重做
