## Pushdozer — Simple, powerful terrain editing for Minecraft

Pushdozer is a Fabric mod for large-scale terrain editing. Multiple work modes, 8 brush geometries, height control, undo/redo, and multiplayer sync.

### Features
- **8 work modes**: Excavate, Place, Smooth, Surface Roughen, Surface Convert, Bone Meal, Batch Plant, Shoreline Process
- **8 brush geometries**: Sphere, Box, Octahedron, Cylinder, Cone, Ellipsoid, Tetrahedron, Triangular Prism
- **4 height modes**: No Limit, Follow Player, Lock Height, Custom
- **4 preview modes**: Wireframe, Point Cloud, Surface (translucent faces), None
- **Undo/Redo**: 30 steps per player
- **Multiplayer**: Server-authoritative with batch sync

### Requirements
- Minecraft **1.21.11** (Fabric)
- Fabric Loader ≥ 0.18.2, Fabric API
- Java **21**
