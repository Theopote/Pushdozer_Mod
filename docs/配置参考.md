# Pushdozer 配置参考

## 配置文件位置

Pushdozer 的配置文件保存在：
```
.minecraft/config/pushdozer_config.json
```

## 基础配置项

### 工作模式 (workMode)
```json
{
  "workMode": "EXCAVATE"
}
```

**可选值**：
- `EXCAVATE` - 挖掘模式
- `PLACE` - 铺设模式
- `SMOOTH` - 平滑模式
- `SURFACE_ROUGHEN` - 表面粗糙
- `SURFACE_CONVERT` - 表层转换
- `BONE_MEAL` - 骨粉生长
- `BATCH_PLANT` - 批量种植
- `SHORELINE_PROCESS` - 水岸处理

### 几何体类型 (geometryType)
```json
{
  "geometryType": "SPHERE"
}
```

**可选值**：
- `SPHERE` - 球体
- `BOX` - 长方体
- `OCTAHEDRON` - 正八面体
- `CYLINDER` - 圆柱体
- `CONE` - 圆锥体
- `ELLIPSOID` - 椭球体
- `TETRAHEDRON` - 正四面体
- `TRIANGULAR_PRISM` - 三棱柱

### 最大操作距离 (maxOperationDistance)
```json
{
  "maxOperationDistance": 50
}
```

**范围**：1-100
**说明**：设置工具的最大操作距离

### 显示模式 (displayMode)
```json
{
  "displayMode": "WIREFRAME"
}
```

**可选值**：
- `NONE` - 不显示
- `WIREFRAME` - 线框
- `POINT_CLOUD` - 点云

## 标高配置

### 标高模式 (heightMode)
```json
{
  "heightMode": "NO_LIMIT"
}
```

**可选值**：
- `NO_LIMIT` - 标高不限
- `FOLLOW_PLAYER` - 跟随玩家标高
- `LOCKED_ONCE` - 锁定玩家标高
- `CUSTOM` - 自定义标高

### 锁定标高 (lockedHeight)
```json
{
  "lockedHeight": 64
}
```

**说明**：当标高模式为 `CUSTOM` 时使用

## 工作模式配置

### 挖掘模式配置
```json
{
  "layeredExcavation": false,
  "excavationLayers": 1
}
```

**参数说明**：
- `layeredExcavation` - 是否启用分层挖掘
- `excavationLayers` - 挖掘层数

### 铺设模式配置
```json
{
  "placeMode": "ADAPTIVE_BIOME",
  "selectedNaturalBlockId": "minecraft:grass_block"
}
```

**参数说明**：
- `placeMode` - 铺设模式（`ADAPTIVE_BIOME` 或 `NATURAL_BLOCK`）
- `selectedNaturalBlockId` - 选择的自然方块ID

### 平滑模式配置
```json
{
  "smoothStrength": 0.5
}
```

**参数说明**：
- `smoothStrength` - 平滑强度（0.0-1.0）

### 表面粗糙配置
```json
{
  "roughnessStrength": 0.3,
  "noiseFrequency": 0.1,
  "noisePersistence": 0.5,
  "noiseOctaves": 4,
  "noiseAutoScale": true
}
```

**参数说明**：
- `roughnessStrength` - 粗糙强度（0.0-1.0）
- `noiseFrequency` - 噪声频率
- `noisePersistence` - 噪声持久性
- `noiseOctaves` - 噪声八度数
- `noiseAutoScale` - 自动缩放

### 表层转换配置
```json
{
  "surfaceBlocks": [
    {
      "blockId": "minecraft:grass_block",
      "percentage": 60
    },
    {
      "blockId": "minecraft:dirt",
      "percentage": 30
    },
    {
      "blockId": "minecraft:stone",
      "percentage": 10
    }
  ]
}
```

**参数说明**：
- `surfaceBlocks` - 表层方块列表
- `blockId` - 方块ID
- `percentage` - 出现百分比

### 批量种植配置
```json
{
  "plantType": "TREES",
  "plantDensity": 0.7,
  "clusterScale": 0.5,
  "respectBiomes": true,
  "treeSpecies": "BIOME_ADAPTIVE",
  "flowerGroup": "BIOME_ADAPTIVE",
  "customPlantBlockIds": []
}
```

**参数说明**：
- `plantType` - 种植类型（`TREES`, `FLOWERS`, `GRASS`, `CUSTOM`）
- `plantDensity` - 种植密度（0.0-1.0）
- `clusterScale` - 团簇大小（0.0-1.0）
- `respectBiomes` - 是否遵守生物群系规则
- `treeSpecies` - 树种选择
- `flowerGroup` - 花组选择
- `customPlantBlockIds` - 自定义植物列表

### 水岸处理配置
```json
{
  "shorelineType": "BEACH",
  "shorelineWidth": 5,
  "plantVegetationEnabled": true,
  "vegetationDensity": 0.6,
  "shorelineHeightAbove": false,
  "shorelineHeightBelow": false
}
```

**参数说明**：
- `shorelineType` - 水岸类型（`BEACH`, `EMBANKMENT`, `ADAPTIVE`, `MUDDY`, `ROCKY`）
- `shorelineWidth` - 水岸宽度
- `plantVegetationEnabled` - 是否启用植物种植
- `vegetationDensity` - 植被密度（0.0-1.0）
- `shorelineHeightAbove` - 是否在标高以上操作
- `shorelineHeightBelow` - 是否在标高以下操作

{
  "maxOperationDistance": 50,

## 注意事项

1. **配置文件格式**：必须使用有效的JSON格式
2. **数值范围**：确保数值在有效范围内
3. **方块ID**：使用正确的Minecraft方块ID
4. **备份配置**：修改前建议备份原配置文件
5. **重启游戏**：某些配置修改需要重启游戏生效

## 故障排除

### 配置文件损坏
如果配置文件损坏，删除配置文件后重启游戏，会自动生成默认配置。

### 配置不生效
1. 检查JSON格式是否正确
2. 确认配置文件位置正确
3. 重启游戏
4. 检查游戏日志中的错误信息

### 性能问题
1. 降低操作距离
2. 减少噪声八度数
3. 使用线框显示模式
4. 关闭自动缩放功能
