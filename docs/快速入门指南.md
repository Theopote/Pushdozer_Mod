# Pushdozer 快速入门指南

## 🚀 5分钟快速上手

### 第一步：获取工具
1. 进入创造模式
2. 在物品栏的 `Tools` 分组中找到 `推土机`
3. 将推土机放入快捷栏

### 第二步：打开配置界面
1. 手持推土机工具
2. 按 `P` 键打开配置界面

### 第三步：选择工作模式
在配置界面中选择你想要的工作模式：
- **挖掘模式**：清除方块
- **铺设模式**：铺设地表方块
- **平滑模式**：平滑地形
- **批量种植**：种植植物

### 第四步：配置笔刷
1. 选择笔刷形状（推荐：球体或长方体）
2. 调整半径或尺寸
3. 设置操作距离

### 第五步：执行操作
1. 右键点击目标位置
2. 观察效果
3. 如需撤销，按 `Z` 键

## 🎯 常用场景示例

### 场景1：平整地面
1. 选择 **挖掘模式**
2. 选择 **长方体** 笔刷
3. 设置合适的长度、宽度、高度
4. 右键点击地面进行挖掘

### 场景2：种植森林
1. 选择 **批量种植** 模式
2. 选择 **球体** 笔刷
3. 设置种植类型为 **树木**
4. 调整种植密度
5. 右键点击要种植的区域

### 场景3：平滑山坡
1. 选择 **平滑模式**
2. 选择 **球体** 笔刷
3. 调整平滑强度（建议0.5-0.8）
4. 右键点击山坡进行平滑

### 场景4：创建沙滩
1. 选择 **水岸处理** 模式
2. 选择水岸类型为 **沙滩**
3. 调整水岸宽度
4. 右键点击水域边缘

## ⚙️ 重要设置

### 标高系统
- **标高不限**：自由编辑任何高度
- **跟随玩家**：操作标高跟随玩家位置
- **锁定标高**：固定在当前高度操作

### 显示模式
- **线框**：显示笔刷轮廓（推荐）
- **点云**：显示点云预览
- **表面**：半透明面预览
- **隐藏**：不显示预览

### 操作距离
- 建议从小距离开始（10-20）
- 大范围操作可能影响性能

## 🔧 常用快捷键

| 按键 | 功能 |
|------|------|
| `K` | 打开配置界面 |
| `Z` | 撤销操作 |
| `Y` | 重做操作 |

## 💡 使用技巧

### 性能优化
- 避免过大的操作半径
- 使用线框显示模式
- 分批执行大范围操作

### 精确操作
- 使用锁定标高模式进行精确编辑
- 从小半径开始，逐步调整
- 善用撤销功能

### 多人游戏
- 大范围操作前通知其他玩家
- 避免同时进行多个大操作
- 注意服务器性能

## 🆘 常见问题

**Q: 为什么操作没有效果？**
A: 检查是否手持推土机工具，工作模式是否正确设置。

**Q: 如何恢复误删的地形？**
A: 按 `Z` 键撤销操作，或按 `Y` 键重做。

**Q: 游戏卡顿怎么办？**
A: 降低操作半径，关闭不必要的显示模式。
